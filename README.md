### 環境構築 
1. リポジトリのクローン:
   ```bash
   git clone https://github.com/Yanai1005/Brakiocup2025-front.git
   ```
2. プロジェクトディレクトリへの移動:
   ```bash
   cd Brakiocup2025-front
   ```
3. パッケージのインストール:
   ```bash
   npm install
   ```
4. ローカルサーバーを起動
   ```bash
   npm run dev
   ```
